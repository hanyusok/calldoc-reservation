----------공통항목(필수)
파라미터 값 설명
서버 서버환경
TYPE 결제방식(P:PC/M:모바일/W:웹뷰)
PAYMETHOD 결제수단
CPID 상점아이디
ORDERNO 주문번호
PRODUCTTYPE 상품구분(1: 디지털, 2: 실물)
AMOUNT 결제금액
PRODUCTNAME 상품명
PRODUCTCODE 상품코드
USERID 고객 ID
----------공통항목(선택)
EMAIL 고객 이메일
USERNAME 고객명
RESERVEDINDEX1 예약항목
RESERVEDINDEX2 예약항목
RESERVEDSTRING 예약스트링
HOMEURL 결제 성공 후, 이동할 URL(결제 창에서 이동)
DIRECTRESULTFLAG 키움페이 결제 완료 창 없이 HOMEURL로 바로 이동(Y/N)
SET_LOGO 결제 창 하단 상점로고 URL
----------웹뷰결제창(필수)
HOMEURL 공통항목(선택)에 있음 결제 성공 후, 이동할 URL(결제 창에서 이동)
CLOSEURL 결제 창에서 취소 누를 시 이동할 URL
FAILURL 결제실패 후, 확인버튼 입력 시 이동할 URL
APPURL 인증완료 후 돌아갈 앱의 URL(CARD(K), BANK 필수)
----------신용카드(필수)
TAXFREECD 과세/비과세여부(00: 과세, 01: 비과세, 02: 복합과세)
TELNO2 비과세 대상금액(TAXFREECD가 02인 경우에만 필수전송)
----------신용카드(조건부 필수)
BILLTYPE CARD(K)-SUGI, CARD(K)-BATCH, MOBILE-BATCH인 경우 필수 과금 유형(1: 일반, 2: 월자동, 14:카드월자동, 13:수기일반, 18:수기비인증)
CERTTYPE CARD(K)-SUGI인 경우 필수(00:카드번호/유효기간)(11:카드번호/유효기간/생년월일/비밀번호앞2자리)
AUTOINFOFLAG CARD(K)-BATCH인 경우 필수 AUTOKEY 전송방식(A~D타입 메뉴얼참고)
----------신용카드(선택)
CPQUOTA CARD, CARD-SUGI인 경우 최대 할부 개월 수(구분자 “ : “) (일시불~12개월 예: 0:2:3:4~~~:12)
CARDQUOTA CARD, CARD-SUGI인 경우 카드사별 할부 개월 수(예시 : CCBC-0:2:3:4:5:6^CCNH-0:2:3)
QUOTAOPT CARDK, CARDK-SUGI인 경우 최대 할부 개월 수 (일시불~12개월 예: 12)
CARDLIST CARD(K)인 경우 결제 창 카드사 노출 값(구분자 “ : “)(카드사코드는 매뉴얼 하단참고)
HIDECARDLIST CARD(K)인 경우 결제 창 카드사 숨김 값(구분자 “ : “)(카드사코드는 매뉴얼 하단참고)
SKIP_YN 카드사 인증창 바로 이동 Y/N(CARDLIST에 하나의 카드사 코드만)
ENG_FLAG CARDK인 경우 영문 결제창 사용여부 Y/N
OPTIONPARAMS CARD-BATCH인 경우 CARDCODE를 값에 추가하여 AUTOKEY 전송방식 변경 (추가 옵션)
----------휴대폰(선택)
MOBILECOMPANYLIST MOBILE,MOBILE-BATCH 결제 창 통신사만 노출 값(구분자 “ , “)통신사 코드: SKT/KTF/LGT/CJH/KCT
FIXTELNO 휴대폰 번호 고정 입력
----------가상계좌(선택)
DEPOSITENDDATE 입금만료일(YYYYMMDD24MISS)(미지정 기본값 7일)
VTAXFREECD 과세여부(00:과세 / 01:비과세 / 02:복합) RESERVEDSTIRNG : 비과세 대상금액 ^[비과세금액] 형식, 복합일 경우에만 사용
BANKCODE 은행코드(제한하는 경우에만)
----------현금영수증(선택)
CASHRECEIPTFLAG 현금영수증 발행여부(1:발행, 0:미발행)
CASHRECEIPTUSE 현금영수증 발행용도(1:소득공제, 2:지출증빙, 9:미발행)
CASHRECEIPTINFO 현금영수증 발행정보(휴대폰번호, 사업자번호)
----------카카오페이(선택)
FREE_AMT 결제금액 중 비과세 금액
CASHRECEIPTFLAG 현금영수증 발행여부(Y:발행, N:미발행)는 카카오페이 현금영수증 발행은 구분하여 사용바랍니다.
PAYTYPE 결제방식 CARD, MONEY(카드, 카카오머니)
USEKAKAO Y/N(신용카드, 카카오 모두 사용시 신용카드 결제 창에서 카카오 표기 여부)
